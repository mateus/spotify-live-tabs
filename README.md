# Development

#### `.env` (both Server and Client apps)

```.env
REACT_APP_MUSICXMATCH_KEY={MUSICXMATCH_KEY}
REACT_APP_CLIENT_ID={SPOTIFY_APP_CLIENT_ID}
CLIENT_SECRET={SPOTIFY_APP_CLIENT_ID}
```

### Start Server
#### `$ yarn install`
#### `$ yarn start`

### Start Client
#### `$ cd client` 
#### `$ yarn install`
#### `$ yarn start`
